# IDSynth

Live page: [mysticalg.github.io/IDSynth](https://mysticalg.github.io/IDSynth/)

Legacy VST synth by Dave Hook / Metalsoft.

## Features

- Phat 8-osc detune and sync
- Pitch envelope
- FM envelope
- 3 main oscillators
- Classic analog-style filter

## Install

Extract `IDSynth.dll` and copy it into your VST plugins folder.

## Included files

- `IDSynth.dll`
- `VSTI-IDSynth-Beta2.rar`
- Legacy release notes in `ReadMe.txt`
