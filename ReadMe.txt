IDSynth Version 1.0 Beta 
Created by Dave Hook (Metalsoft 2005)


Features:

This synth is a good all-rounder and great for trance leads, bass, pads and sfx.

-Phat 8 osc detune and sync
-Pitch Envelope
-FM envelope
-3 main oscs
-classic analog style filter


Install:

Extract IDSynth.dll and put in into your vsti hosts vstplugins directory.
